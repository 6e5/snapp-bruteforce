Snapp Bruteforce script
======

Usage
-------
Make sure you have node js >= `8` installed
```sh
git clone https://github.com/amiralies/snapp-bruteforce.git
cd snapp-bruteforce
#if you have yarn
yarn
#or npm
npm install
node app.js [email] [password list file path]
```
Example
-------
```sh
node app.js example@example.com pw.txt
```
